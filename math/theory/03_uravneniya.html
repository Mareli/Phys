3. Рациональные уравнения

Уравнения с дробями: приведение к общему знаменателю, ОДЗ и проверка посторонних корней.

ТеорияОДЗ обязательна
⚠️
Главное правило При умножении обеих частей на знаменатель могут появиться посторонние корни. Обязательно проверь каждый найденный корень в ОДЗ!
Алгоритм решения дробного рационального уравнения
1. Найти ОДЗ: Q(x) ≠ 0 и S(x) ≠ 0. 2. Умножить на НОЗ. 3. Решить целое уравнение. 4. Исключить из ответа значения вне ОДЗ.

4. Уравнения с модулями

Раскрытие модуля по определению и решение методом разбора случаев.

ТеорияЦТ-классика
Определение модуля
. Геометрически: расстояние от a до 0 на числовой прямой.
|f(x)| = a, где a ≥ 0
При a < 0 решений нет
|f(x)| = g(x)
Необходимое условие: g(x) ≥ 0

Метод раскрытия по определению

💡
Алгоритм: 1) Найти «критические» точки, где выражение под модулем обращается в 0. 2) Разбить числовую прямую на промежутки. 3) На каждом промежутке раскрыть модуль без знака (+ или −). 4) Решить на каждом промежутке. 5) Объединить ответы.

6. Системы уравнений

Метод подстановки, метод сложения/вычитания, графический метод. Системы с нелинейными уравнениями.

ТеорияВажно

Метод подстановки

1. Выразить одну переменную через другую из одного уравнения.

2. Подставить в другое уравнение.

3. Решить полученное уравнение с одной переменной.

Метод сложения

1. Умножить уравнения на подходящие числа.

2. Сложить уравнения так, чтобы одна переменная сократилась.

3. Решить и найти вторую переменную.

Система линейных уравнений
Решается подстановкой или методом сложения

Системы с нелинейными уравнениями

💡
Типичная система на ЦТ: одно уравнение линейное, другое — квадратное. Из линейного выражаем y = f(x), подставляем в квадратное. Получаем квадратное уравнение относительно x.
Симметричная система
Сумма и произведение — сразу применяем теорему Виета

Геометрический смысл системы

Геометрическая интерпретация
Решение системы — это координаты точек пересечения графиков обоих уравнений. Нет решений → графики не пересекаются. Бесконечно много → графики совпадают.
